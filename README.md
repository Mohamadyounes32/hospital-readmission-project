# 🏥 Hospital Readmission Analysis

Healthcare analytics project focused on analyzing hospital readmissions using Python, SQL, statistical analysis, and Power BI.

---

# 📌 Project Overview

Hospital readmissions are one of the biggest challenges in healthcare systems.

This project analyzes patient hospitalization data to identify:
- Readmission patterns
- High-risk patient groups
- Clinical drivers of readmission
- Length of stay trends
- Utilization behavior

The project also includes an interactive Power BI dashboard with scenario simulation capabilities.

# 📊 Analysis Workflow

## 1. Data Cleaning
- Handled missing values
- Removed duplicated records
- Standardized categorical variables
- Prepared clean dataset for analysis

## 2. Exploratory Data Analysis (EDA)
- Readmission distribution analysis
- Length of stay analysis
- Medication usage analysis
- Correlation analysis
- Specialty-level analysis

## 3. Statistical Analysis
- Mann-Whitney U Test
- Chi-Square Test
- Statistical validation of readmission patterns

## 4. Feature Engineering
Created new features such as:
- Risk Category
- Utilization Score
- Readmission Binary Flag

## 5. SQL Analysis
Performed SQL-based healthcare analytics queries:
- Readmission rates
- Risk segmentation
- Specialty-level insights
- Utilization ranking

## 6. Dashboard Development
Developed an interactive Power BI dashboard with:
- Executive overview
- Clinical insights
- Risk analysis
- Scenario simulation

---

# 📈 Key Insights

- High-risk patients showed significantly higher readmission rates.
- Longer hospital stays were associated with higher early readmission probability.
- Certain specialties such as nephrology and oncology exhibited elevated readmission patterns.
- Patients with higher utilization scores demonstrated increased healthcare burden.
